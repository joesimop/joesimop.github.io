This program was a part of my CPE 357 class, and I enjoyed it so much! It's effectivley an object oriented implementation of a Hash Table in C. By using dynamic memory allocation and custom structs, the Hash Table uses separate chaining to create an quick way of storing data. When creating the Hash Table, the user has the ability to put in their own functions in case they want to store custom Structs in the Hash Table. This way they can destory and hash as they please. If they don't want to put in their own functions, I wrote some basic ones for them! The implementation also has some level of protection, as the user can only access variables that I wrote functions for them to access.
